## Микросервисы

### 1. Catalog Service — управление товарами

| Артефакт              | Файл                              |
|-----------------------|-----------------------------------|
| Структура БД          | [db/catalog_db.md](db/catalog_db.md) |
| API (Postman)         | [api/catalog_api.json](api/catalog_api.json) |

**Основные эндпоинты:**

| Метод  | Путь                         | Описание                        |
|--------|------------------------------|---------------------------------|
| GET    | /api/v1/categories           | Список категорий                |
| GET    | /api/v1/products             | Список товаров (с фильтрами)    |
| GET    | /api/v1/products/:id         | Карточка товара                 |
| POST   | /api/v1/products             | Создать товар (admin)           |
| PATCH  | /api/v1/products/:id         | Обновить товар (admin)          |
| DELETE | /api/v1/products/:id         | Скрыть товар (soft delete, admin)|

---

### 2. Order Service — управление заказами и корзиной

| Артефакт              | Файл                              |
|-----------------------|-----------------------------------|
| Структура БД          | [db/orders_db.md](db/orders_db.md) |
| API (Postman)         | [api/orders_api.json](api/orders_api.json) |

**Основные эндпоинты:**

| Метод  | Путь                          | Описание                        |
|--------|-------------------------------|---------------------------------|
| GET    | /api/v1/cart                  | Получить корзину                |
| POST   | /api/v1/cart/items            | Добавить товар в корзину        |
| PATCH  | /api/v1/cart/items/:id        | Изменить количество             |
| DELETE | /api/v1/cart/items/:id        | Удалить позицию из корзины      |
| DELETE | /api/v1/cart                  | Очистить корзину                |
| POST   | /api/v1/orders                | Оформить заказ (checkout)       |
| GET    | /api/v1/orders/:id            | Получить заказ                  |
| GET    | /api/v1/orders                | Список заказов (admin)          |
| PATCH  | /api/v1/orders/:id/status     | Изменить статус (admin)         |

**Жизненный цикл заказа:**

```
new → confirmed → processing → shipped → delivered
                                        ↘ cancelled (из любого статуса кроме delivered)
```

### 3. Admin Service — панель управления

| Артефакт              | Файл                              |
|-----------------------|-----------------------------------|
| Структура БД          | [db/admin_db.md](db/admin_db.md)  |
| API (Postman)         | [api/admin_api.json](api/admin_api.json) |

**Основные эндпоинты:**

| Метод  | Путь                            | Описание                         |
|--------|---------------------------------|----------------------------------|
| POST   | /api/v1/admin/auth/login        | Вход в панель управления         |
| POST   | /api/v1/admin/auth/logout       | Выход                            |
| GET    | /api/v1/admin/auth/me           | Текущий администратор            |
| GET    | /api/v1/admin/users             | Список администраторов           |
| POST   | /api/v1/admin/users             | Создать администратора           |
| PATCH  | /api/v1/admin/users/:id         | Изменить роль / деактивировать   |
| GET    | /api/v1/admin/products          | Все товары (вкл. неактивные)     |
| PATCH  | /api/v1/admin/products/:id/stock| Обновить остаток на складе       |
| GET    | /api/v1/admin/audit-log         | Журнал действий администраторов  |

**Роли администраторов:**

| Роль          | Права                                              |
|---------------|----------------------------------------------------|
| super_admin   | Полный доступ: пользователи, товары, заказы, аудит |
| manager       | Управление товарами и заказами                     |
| warehouse     | Просмотр товаров, обновление остатков              |
